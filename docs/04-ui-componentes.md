# 4. UI e 5. Componentes

## Sistema visual
- Cantos discretos: `rounded-lg`.
- CTAs em azul royal.
- Cards brancos com borda fina.
- Tipografia forte em Manrope para headlines.
- Inter para leitura e interface.

## Componentes principais
- `Header`: navegacao, login, favoritos e carrinho.
- `Footer`: links institucionais, contato e WhatsApp.
- `StoreProvider`: estado de carrinho e favoritos.
- `CartDrawer`: carrinho lateral animado.
- `ProductCard`: card reutilizavel de produto.
- `ProductDetails`: galeria, seletores e compra.
- `HomeSections`: hero, beneficios, produtos, categorias, banner e avaliacoes.
