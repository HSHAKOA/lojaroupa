# 9. Deploy

## Vercel
1. Subir o repositorio para GitHub.
2. Importar projeto na Vercel.
3. Configurar variaveis de ambiente.
4. Rodar build automatico.

## Netlify
1. Configurar build command: `npm run build`.
2. Configurar publish directory: `.next`.
3. Adicionar plugin oficial do Next.js se necessario.
4. Configurar variaveis de ambiente.

## Checklist
- `npm install`
- `npm run build`
- Configurar Stripe.
- Configurar dominio.
- Ativar analytics.
- Validar mobile e performance.
